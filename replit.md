# Qirox Cafe

## Overview
A single-page cafe website for Qirox Cafe, showcasing the menu, about section, hours, testimonials, and contact form.

## Project Architecture
- **Tech Stack**: Vanilla HTML/CSS/JS served with `serve` (Node.js)
- **Structure**:
  - `public/` - Static files served to the browser
    - `index.html` - Main page
    - `css/styles.css` - All styles
    - `js/main.js` - Interactive features (menu tabs, nav, contact form)
    - `assets/` - Stock images (hero, coffee, pastry, sandwich, about)
- **Port**: Frontend served on 0.0.0.0:5000
